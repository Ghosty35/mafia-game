# Mafia Game 2026 - Player Guide & System Documentation Index

This folder contains detailed, up-to-date documentation for every major system.

Use these as source for:
- In-game help text
- Tooltips
- Loading screen tips
- Official wiki / guide
- Admin reference

## Files

- **SYSTEMS_OVERVIEW.md** - High level overview of all systems
- **ECONOMY_TAX_SYSTEM.md** - All tax rates, government fund, admin controls, ratios
- **CRIME_SYSTEM.md** - Success rates, rewards, XP rules, heat, donator bonuses
- **FAMILY_SYSTEM.md** - Creation, power, hourly pay formula (60/40), roles, VIP
- **SAFEHOUSE_PROPERTIES.md** - Limits, naming, shed upgrades, piggybank, greetings
- **WEED_GROWING.md** - Quality %, harvest calculation, carry caps
- **BANKING_SYSTEMS.md** - Personal, Family, Piggybank, taxes on transfers
- **RACING_SYSTEM.md** - 2-player rules, timers, entry fees, car health
- **JAIL_DEATH_SYSTEM.md** - Jail training, death timer, breakout
- **VEHICLE_GARAGE_SYSTEM.md** - Health, tuning parts, categories, images
- **DONATOR_VIP.md** - All current perks and future ideas
- **ADMIN_TOOLS.md** - What admins can do + special YGhosty setup
- **LIVE_TRACKERS.md** - All timers, consistency rules
- **PLAYER_PROGRESSION.md** - Ranks, XP, rebirth, skills
- **REAL_ESTATE_BILLING.md** - Purchase tax, weekly bills, Post Office
- **STREET_DEALER_DRUGS.md** - Buy/sell, city prices, carry caps

## How to Use
1. Keep these files updated after every major change.
2. When adding new features, create or update the relevant .md
3. Extract short helpful tips from these for in-game UI (tooltips, help buttons, Post Office, etc.)
