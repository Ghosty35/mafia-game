# Admin Tools

## Access
- Only for username "YGhosty" (or players with admin flag)
- Full no-restriction access in game

## Features
- Give money to any player (persists to DB)
- Change tax rates (Properties, Bank, etc.)
- View Admin Logbook (activity, promotions, kills, heists)
- Simulated ban/kick/assign moderator
- Economy / Tax summary (total collected, earnings)

## Special for YGhosty
- Auto set on load:
  - 4,000,000 cash
  - 50,000 power
  - Godfather level
  - 75% KillSkill
  - Donator status

## Tax Control
- Admin can set global rates per category (0-100%)
- Changes affect all players immediately (via DB/config)
