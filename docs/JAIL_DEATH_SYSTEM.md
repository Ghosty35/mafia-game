# Jail & Death System

## Death
- When health reaches 0 during crime/heist
- Player is dead for 60 minutes
- Limited access (mainly leaderboard + jail training)
- Force respawn available for testing (admin or force button)

## Jail
- Separate from death
- Train breakout skill (costs money, increases %)
- Attempt breakout based on skill vs random
- See other inmates (demo list for now)
- Part of Safehouse / Post Office flow in future

## Interaction with Other Systems
- Death prevents most actions
- Jail allows training even while "dead" in some cases
- Breakout skill persists on player
