# Racing System

## Core Rules
- Must be 2 players (no solo)
- Post a race with timer (5min / 15min / 1h / 2h)
- Both pay entry fee (balanced ratio, e.g. 10% of bet)
- Choose car with min 75% health
- Winner takes the pot

## Posting a Race
- Select car (from garage)
- Set bet amount
- Choose expire time
- Pay entry fee upfront

## Joining & Starting
- Opponent joins before timer ends
- If no one joins → auto cancel + message
- If joined → can start race
- Both get message in /messages when timer ends

## Vehicle Health
- Cars have health (condition)
- Below 75% cannot be used for racing
- Repair and tuning in Garage affect this

## Cooldown
- 10 minutes after every race (live tracker)
- Applies to both winner and loser

## History
- Shows past races: winner vs loser, amount won
- Current open challenges with live timers

## Balance
- Entry fee prevents spam
- Health requirement makes garage/tune shop useful
- Timer options allow different playstyles
